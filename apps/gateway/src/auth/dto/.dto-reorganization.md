### 2. 统一导出

创建了 `dto/index.ts` 统一导出所有 DTO：

```typescript
// 方式一：统一导入（推荐）
import { CreateUserDto, LoginDto } from "./dto";

// 方式二：单独导入
import { CreateUserDto } from "./dto/auth.dto";
```
