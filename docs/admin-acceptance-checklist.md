## 1. 功能验收

### 1.1 用户管理（5 个功能）

#### 列出用户

- [ ] **GET /admin/users** - 正常返回用户列表
  - [ ] 支持搜索（searchValue）
  - [ ] 支持分页（limit/offset）
  - [ ] 支持排序（sortBy/sortDirection）
  - [ ] 返回正确的用户信息（id, email, role, banned等）
  - [ ] 响应时间 < 100ms

#### 获取用户详情

- [ ] **GET /admin/users/:id** - 正常返回用户详情
  - [ ] 返回完整的用户信息
  - [ ] 包含封禁信息（banned, banReason等）
  - [ ] 响应时间 < 50ms

#### 创建用户

- [ ] **POST /admin/users** - 正常创建用户
  - [ ] 必填字段验证（email, password）
  - [ ] 可选字段支持（name, role, emailVerified）
  - [ ] 密码加密存储
  - [ ] 响应时间 < 50ms

#### 更新用户

- [ ] **PUT /admin/users/:id** - 正常更新用户
  - [ ] 支持更新所有字段
  - [ ] 权限验证（管理员权限）
  - [ ] 响应时间 < 50ms

#### 删除用户

- [ ] **DELETE /admin/users/:id** - 正常删除用户
  - [ ] 权限验证（超级管理员权限）
  - [ ] 级联删除相关数据
  - [ ] 响应时间 < 50ms

### 1.2 角色与权限（2 个功能）

#### 设置角色

- [ ] **POST /admin/users/:id/role** - 正常设置角色
  - [ ] 支持 3 种角色（user, admin, superadmin）
  - [ ] 权限提升保护（仅超级管理员可设置 superadmin）
  - [ ] 角色变更日志
  - [ ] 响应时间 < 30ms

#### 检查权限

- [ ] **POST /admin/check-permission** - 正常检查权限
  - [ ] 支持细粒度权限检查
  - [ ] 返回准确的权限结果
  - [ ] 响应时间 < 10ms

### 1.3 用户状态管理（2 个功能）

#### 封禁用户

- [ ] **POST /admin/users/:id/ban** - 正常封禁用户
  - [ ] 支持封禁原因（banReason）
  - [ ] 支持过期时间（banExpiresIn）
  - [ ] 封禁用户无法登录
  - [ ] 响应时间 < 50ms

#### 解封用户

- [ ] **POST /admin/users/:id/unban** - 正常解封用户
  - [ ] 清除封禁状态
  - [ ] 用户可正常登录
  - [ ] 响应时间 < 50ms

### 1.4 会话管理（2 个功能）

#### 列出会话

- [ ] **GET /admin/users/:id/sessions** - 正常列出会话
  - [ ] 返回所有活跃会话
  - [ ] 包含会话详情（ipAddress, userAgent等）
  - [ ] 响应时间 < 100ms

#### 撤销会话

- [ ] **POST /admin/sessions/:token/revoke** - 正常撤销会话
  - [ ] 会话立即失效
  - [ ] 用户被强制登出
  - [ ] 响应时间 < 50ms

### 1.5 用户模拟（2 个功能）

#### 模拟用户

- [ ] **POST /admin/impersonate/:id** - 正常模拟用户
  - [ ] 权限验证（管理员权限）
  - [ ] 生成模拟会话
  - [ ] 审计日志记录
  - [ ] 响应时间 < 50ms

#### 停止模拟

- [ ] **POST /admin/stop-impersonating** - 正常停止模拟
  - [ ] 恢复原始会话
  - [ ] 清除模拟会话
  - [ ] 响应时间 < 30ms
