## 🎯 用户流程验证

### 完整注册流程

```
/register → 填写表单 → 提交
  ↓
显示"查收验证邮件"提示
  ↓
用户点击邮件链接
  ↓
/verify-email?token=xxx → 自动验证
  ↓
验证成功 → 3 秒后跳转 /login
```

### 完整登录流程

```
/login → 填写表单 → 提交
  ↓
验证通过 → 跳转 /dashboard
```

### 密码重置流程

```
/forgot-password → 输入邮箱 → 提交
  ↓
显示"查收重置邮件"提示
  ↓
用户点击邮件链接
  ↓
/reset-password?token=xxx → 输入新密码
  ↓
重置成功 → 跳转 /login
```
