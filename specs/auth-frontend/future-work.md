### 常见用户问题

1. **Q: 为什么登录失败后没有错误提示？**
   **A:** 已修复，现在会显示具体的错误信息（如邮箱或密码错误）

2. **Q: 2FA 备用码可以重新生成吗？**
   **A:** 暂不支持，计划在 v1.5 版本添加此功能

3. **Q: 支持哪些 OAuth 提供商？**
   **A:** 目前支持 GitHub 和 Google，未来计划支持更多
