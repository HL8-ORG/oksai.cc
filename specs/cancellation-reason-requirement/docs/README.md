# 取消原因必填规则文档

## 概述

该功能允许 Event Type 所有者配置取消原因在何种情况下为必填。

## 截图

功能实现后会在此处补充截图。

## 配置

该设置位于 Event Type → Advanced Settings，在 Booking Questions 区块之后。

### 选项

- **Mandatory for both**：host 与 attendee 在取消时都必须填写原因
- **Mandatory for host only**（默认）：仅 host 必须填写原因
- **Mandatory for attendee only**：仅 attendee 必须填写原因
- **Optional for both**：所有人都可选填写取消原因
