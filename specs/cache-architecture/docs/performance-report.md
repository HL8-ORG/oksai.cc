## 生产环境建议

### 配置建议

```typescript
{
  // L1 配置
  l1: {
    max: 10000,        // 生产环境建议 10000
    ttl: 30000,        // 30 秒
    enableStats: true,
  },

  // L2 配置
  l2: {
    enabled: true,
    type: 'single-node', // 或 'cluster'
    ttl: 7200000,        // 2 小时
    keyPrefix: 'oksai:',
  },

  // TTL 抖动
  jitter: {
    enabled: true,
    variance: 0.1,      // ±10%
  },
}
```

### 监控指标

1. **关键指标**:
   - L1/L2 命中率（应 > 80%/90%）
   - 平均响应时间（L1 < 1ms, L2 < 10ms）
   - In-flight request 数量
   - 缓存大小

2. **告警阈值**:
   - 总体命中率 < 80%
   - L2 响应时间 > 50ms
   - In-flight request 数量 > 100
